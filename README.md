# 🍄 Klasifikasi Jamur (KNN)

Project Machine Learning untuk mengklasifikasikan kualitas jamur menjadi Grade A, B, dan C menggunakan metode K-Nearest Neighbor (KNN).

## 📌 Fitur
- Preprocessing data
- Training model KNN
- Evaluasi akurasi
- Prediksi kualitas jamur

## 🛠️ Teknologi
- Python
- Scikit-Learn
- Pandas
- NumPy

## 📊 Metode
Algoritma yang digunakan adalah K-Nearest Neighbor (KNN) untuk klasifikasi berdasarkan fitur numerik.

## 🚀 Cara Menjalankan
```bash
pip install -r requirements.txt
python main.py
